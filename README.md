Installation
------------
Copy install.sh and /usr to minix.
Run chmod +x install.sh
Run ./install.sh
	This assumes your system runs bash-2.05
cd /usr/src/
make libraries
make includes
cd servers/
make install
cd /usr/tools/
make install
restart your system
